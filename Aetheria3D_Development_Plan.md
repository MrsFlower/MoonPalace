# Aetheria3D (VoxStory Engine) 开发计划

## 概述
基于您提供的项目介绍与 `2D_TO_3D_EXPERIENCE.md` 中的经验提炼，本项目（基于生成式AI与RayLib库的3D游戏场景自动化构建系统）架构庞大且涉及跨语言（Python/Moonbit/C）及多模态模型集成。为有效控制开发复杂度，防范底层 FFI 指针、模型生成不可控等风险，特制定分阶段、测试驱动的开发计划。

## 阶段一：数据契约与基础环境搭建 (The JSON Bridge & Env)
**目标**：确立端到端的数据标准，跑通 Moonbit 与 Raylib 的最简绑定。
- **任务 1.1**：定义 `world_data.json` 数据规范（Schema），包含坐标系映射、碰撞体尺寸、剧情触发节点。
- **任务 1.2**：搭建 Python 端基础脚手架，实现基础的 JSON 校验逻辑。
- **任务 1.3**：搭建 Moonbit 项目，完成与 Raylib 的极简 C-FFI 绑定（窗口初始化、主循环、基础摄像机）。
- **单元测试**：
  - [Python] JSON Schema 严格校验测试。
  - [Moonbit] 内存安全的 FFI 窗口创建与销毁测试。

## 阶段二：Moonbit 客户端核心渲染与物理层 (The Moonbit Client)
**目标**：在不依赖 AI 生成数据的情况下，使用 Mock 数据跑通 3D 漫游与碰撞。
- **任务 2.1**：实现 Moonbit 侧的健壮 JSON 解析器（参考 2D 经验，防止超大 JSON 解析截断）。
- **任务 2.2**：封装 Raylib 3D 渲染接口，支持加载并渲染静态点云/体素网格。
- **任务 2.3**：实现底层碰撞系统，将提取的“隐形体素网格”作为 Collider，处理实体碰撞。
- **任务 2.4**：构建 Moonbit 核心游戏状态机（如 `Init` -> `Loading` -> `Roam` -> `Dialog`）。
- **单元测试**：
  - [Moonbit] 状态机非法流转拦截测试。
  - [Moonbit] 边界碰撞检测算法测试（AABB 包围盒交集）。

## 阶段三：AI 编译管线 - 逻辑与视觉生成 (The AI Compiler - Part 1)
**目标**：打通文本到 2D 资产与剧情数据的自动化流程。
- **任务 3.1**：构建 LLM 提示词工程（两阶段策略）：阶段一出场景大纲，阶段二出结构化 `world_data.json`。
- **任务 3.2**：集成 SDXL + ControlNet 脚本，传入场景描述与严格的等距视角（Isometric）/深度图约束，生成 2D 图像。
- **单元测试**：
  - [Python] LLM 输出不稳定情况下的重试与 Fallback 机制测试。
  - [Python] 图像生成 API 的容错拦截与日志捕获测试。

## 阶段四：AI 编译管线 - 3DGS 生成与空间降维 (The AI Compiler - Part 2)
**目标**：实现从 2D 图到 3D 点云，再到离散碰撞数据的转换。
- **任务 4.1**：集成 DiffusionGS，将 2D 图像极速重建为 `.ply` 3DGS 点云文件。
- **任务 4.2**：开发“强制离散化”后处理 Python 脚本，基于空间不透明度阈值，将连续点云采样为离散的体素矩阵/包围盒网格（`.obj` 或自定义格式）。
- **单元测试**：
  - [Python] 体素化采样算法的边界值测试（如极低密度点云、异常坐标系点云）。

## 阶段五：端到端集成与 UI 交互 (Integration & Polish)
**目标**：联调 Python 端输出与 Moonbit 端输入，实现完整的游玩闭环。
- **任务 5.1**：端到端管线串联：文本 -> Python 管线 -> 资产文件夹 -> Moonbit 引擎自动加载。
- **任务 5.2**：实现 Raylib 2D UI 叠加层，处理玩家触碰剧情节点后的对话框弹出与交互。
- **任务 5.3**：引入全局异步加载与加载过渡动画（防卡顿/防掉帧）。
- **集成测试**：
  - 测试 Python 生成的资产包能否被 Moonbit 客户端 100% 无损解析、渲染不穿模、剧情节点坐标映射正确。

## 阶段六：系统健壮性与文档输出 (Docs & Release)
**目标**：提升系统稳定性，交付标准文档。
- **任务 6.1**：完善跨语言日志系统（Log System），在 Moonbit 端增加可视化的 Debug 面板。
- **任务 6.2**：编写系统部署手册（云端 GPU 容器 + 本地 Moonbit）。
- **任务 6.3**：完善 API 文档与 JSON 数据结构规范说明。
